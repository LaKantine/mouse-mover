# Mouse Mover v1.0 (Mac)

Ce programme bouge la souris aléatoirement toutes les 5 secondes.

## Installation

1. Téléchargez `mouse_mover_mac.app.zip` depuis GitHub Release.
2. Décompressez pour obtenir `mouse_mover.app`.
3. Double-cliquez pour lancer.
4. Appuyez sur **ESPACE** pour arrêter le programme.

## Notes

- Nécessite macOS.
- Aucun failsafe actif. Soyez prudent.
